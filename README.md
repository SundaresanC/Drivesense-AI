# 🚗 DriveSense AI

**Real-time AI driving companion using Vision-Agents framework**

An intelligent assistant that watches the road, detects threats (pedestrians, vehicles, lane drift), and speaks safety guidance to drivers. Designed to assist visually impaired and elderly drivers.

This is a production-ready implementation using the [GetStream Vision-Agents](https://github.com/GetStream/Vision-Agents) framework for real-time, low-latency AI-powered driving assistance.

---

## 🎯 Project Overview

DriveSense AI is an intelligent driving companion that:

- **👁 Watches the road** via camera in real-time (< 30ms latency)\n- **🎧 Listens to driver voice** for context-aware guidance\n- **🧠 Understands traffic** with object detection, lane analysis, and depth estimation\n- **🗣 Speaks natural guidance** via high-quality text-to-speech\n- **📊 Provides context** about pedestrians, vehicles, traffic signs, and lane position\n- **⚡ Responds immediately** to emergencies\n\n### 💡 Example Interactions\n\n**Driver:** \"Can I turn right?\"\n**Agent:** \"Do not turn right. There is a pedestrian crossing from that direction. Wait.\"\n\n**Scene:** Pedestrian ahead\n**Agent:** \"Pedestrian in center. Prepare to brake.\"\n\n**Scene:** Drifting left\n**Agent:** \"You are drifting left. Adjust steering.\"\n\n---\n\n## 🚀 Features\n\n### ✅ Core Capabilities\n- [x] Real-time object detection (YOLO)\n- [x] Lane detection & drift warnings\n- [x] Distance estimation to objects\n- [x] Traffic sign recognition\n- [x] Pedestrian/vehicle threat assessment\n- [x] Voice-based driver queries\n- [x] Natural language guidance\n- [x] Multi-modal reasoning (vision + audio)\n- [x] Low-latency processing\n\n### 🔧 Technical Stack\n\n| Component | Technology | Purpose |\n|-----------|-----------|----------|\n| **Vision** | YOLOv8 + Ultralytics | Real-time object detection |\n| **Lane Detection** | OpenCV | Lane marking identification |\n| **Reasoning** | Gemini / OpenAI LLM | Intelligent decision making |\n| **Voice Input** | Deepgram / Whisper | Speech-to-text |\n| **Voice Output** | ElevenLabs | Natural text-to-speech |\n| **Framework** | Vision-Agents | Production-ready agent framework |\n| **Infrastructure** | Stream.io | Ultra-low latency edge network |\n\n---\n\n## 📋 Prerequisites\n\n### System Requirements\n- **OS**: Windows / macOS / Linux\n- **Python**: 3.9+\n- **Camera**: Webcam or IP camera\n- **RAM**: 8GB+ recommended\n- **GPU**: NVIDIA GPU (optional, for faster processing)\n\n### API Keys (Optional but Recommended)\n\n1. **Gemini API** (Google): For LLM-based reasoning\n   -
